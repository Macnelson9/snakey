### Description

> **Buga** (Hausa: _to play_) — the Nokia Xenzia snake game, reimagined as a reward-bearing Progressive Web App on Celo.

Buga is an open-source, browser-native retro arcade platform built on the Celo blockchain. Players earn **G$ (GoodDollar tokens)** for verified gameplay — no app download, no signup, just open and play. The game faithfully recreates the classic Nokia Xenzia experience with a dot-matrix LCD aesthetic, 8 selectable themes, D-pad and swipe controls, and a Web Audio sound engine tuned to the original's feel.

**How it works:** Every run is seeded server-side with a cryptographically random seed. When the run ends, the server replays the exact game from inputs to produce an authoritative score — the client can never fake a result. The server then checks whether the player's wallet address is GoodDollar-verified on-chain via the GoodDollar Identity contract. If verified, it signs an EIP-712 reward voucher and calls `GameRewards.redeem()` on Celo mainnet, sending G$ directly to the player's embedded wallet. Unverified players are prompted to complete GoodDollar face-verification to start earning.

**Sub-linear reward curve:** rewards scale with `√(score)` and are capped per identity per day, preventing farming and ensuring rewards reach the widest possible group of real humans.

**Coming next:** async same-seed multiplayer races, private rooms, daily global challenge leaderboards, and additional retro games (Tetris, Pac-Man, Bounce) — building Buga into a full GoodDollar-native retro arcade.

Buga is named in honor of the Nokia phone's outsized cultural legacy in West Africa and other emerging markets, where it was often people's first personal computing device. The game is deliberately low-data and PWA-installable for users on constrained connectivity.
